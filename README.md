# 카드현금화 랜딩페이지

Next.js 기반의 카드현금화 업체 랜딩페이지입니다.

## 기술 스택

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS

## 로컬 개발

```bash
npm install
npm run dev
```

개발 서버: http://localhost:3000

## 빌드

```bash
npm run build
npm start
```

## Vercel 배포

### 1. Vercel CLI 설치 (선택사항)

```bash
npm i -g vercel
```

### 2. 배포

```bash
vercel
```

또는 GitHub 연동을 통해 자동 배포할 수 있습니다.

### 3. 도메인 연결

Vercel 대시보드에서 도메인을 추가하고 DNS 설정을 완료하세요.

## 주요 기능

- 반응형 디자인
- 모던한 UI/UX
- 빠른 로딩 속도
- SEO 최적화
